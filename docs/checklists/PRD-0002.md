## Phase 1: MVP (Web + Mobile)

### 1.1 Core Infrastructure
- [ ] CDN 설정 (Akamai/CloudFront)
- [ ] DRM 연동 (Widevine, FairPlay, PlayReady)
- [ ] HLS 트랜스코딩 파이프라인
- [ ] 스토리지 아키텍처

### 1.2 Authentication & Billing
- [ ] GGPass SSO 연동
- [ ] 내부 빌링 API 연동
- [ ] 구독 플랜 관리 UI

### 1.3 Web Platform
- [ ] hls.js 기반 커스텀 플레이어
- [ ] Watch 화면 구현
- [ ] Collections 화면 구현
- [ ] 1080p 스트리밍 테스트
- [ ] 반응형 디자인

### 1.4 Mobile Apps
- [ ] iOS 앱 (Swift + AVPlayer)
- [ ] Android 앱 (Kotlin + ExoPlayer)
- [ ] 앱스토어 심사 제출

### 1.5 VOD & Quick VOD
- [ ] 라이브→VOD 즉시 전환
- [ ] 시청 이력 저장
- [ ] 이어보기 기능
- [ ] 챕터/구간 탐색

### 1.6 Subtitles
- [ ] 20개국 자막 지원
- [ ] 영어 기반 번역 워크플로우
- [ ] 자막 싱크 검증
- [ ] VOD 자막 품질 검수
