## 5. API 엔드포인트

### 5.1 활성 API
- [ ] **`GET /api/v1/jellyfin/contents`** - 콘텐츠 목록
- [ ] **`GET /api/v1/jellyfin/contents/{id}`** - 콘텐츠 상세
- [ ] **`GET /api/v1/jellyfin/stream/{id}`** - 스트림 URL
- [ ] **`GET /api/v1/jellyfin/libraries`** - 라이브러리 목록
- [ ] **`GET /api/v1/search`** - 검색 (MeiliSearch)
- [ ] **`POST /api/v1/auth/login`** - 로그인
