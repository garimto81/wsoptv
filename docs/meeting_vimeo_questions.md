## 질문 목록 (12개)

### 플레이어/SDK (3개)

#### Q1. Vimeo Player SDK 커스터마이징 범위는? 멀티뷰 레이아웃 직접 구현 가능한가요?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | WSOPTV의 핵심 기능인 멀티뷰를 Vimeo 플레이어에서 구현할 수 있는지 확인해야 합니다. |
| 기술 배경 | Vimeo Player는 iframe embed 방식과 Player SDK 방식이 있습니다. SDK 방식은 더 많은 커스터마이징이 가능하지만, 멀티뷰 같은 고급 기능은 별도 개발이 필요할 수 있습니다. |
| 답변 | |

#### Q2. 라이브 스트리밍 성능과 지연 시간은? 스포츠 라이브에 적합한가요?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | Vimeo는 주로 VOD 중심 플랫폼으로 알려져 있습니다. WSOPTV는 라이브 포커 토너먼트가 핵심이므로, Vimeo의 라이브 스트리밍 성능과 대규모 이벤트 처리 경험을 확인해야 합니다. |
| 기술 배경 | 라이브 스트리밍 품질은 지연 시간(Latency), 안정성, 동시접속 처리 능력으로 평가합니다. |
| 답변 | |

#### Q3. 24/7 리니어 채널 구현 방식과 관리 도구는?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | WSOPTV는 24시간 라이브 채널(예: WSOP TV 채널)을 운영할 계획입니다. |
| 기술 배경 | 리니어 채널은 TV처럼 정해진 편성표에 따라 콘텐츠가 자동으로 재생되는 방식입니다. |
| 답변 | |

### OTT/앱 (4개)

#### Q4. 커스텀 앱 생성 시 디자인 커스터마이징 수준은?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | GGPoker는 강력한 브랜드 아이덴티티를 보유하고 있으며, WSOPTV 앱은 이 디자인 가이드를 100% 따라야 합니다. |
| 기술 배경 | Vimeo OTT(Streaming)는 커스텀 브랜드 앱 생성을 제공합니다. "커스텀"의 범위가 색상/로고 변경 수준인지, 완전한 UI 오버라이드가 가능한지 확인이 필요합니다. |
| 답변 | |

#### Q5. TV 앱 지원 플랫폼 범위는?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | WSOPTV는 5개 TV 플랫폼(Samsung, LG, Roku, Fire TV, Apple TV) 전체에서 서비스해야 합니다. |
| 기술 배경 | Vimeo OTT는 템플릿 기반 앱 생성을 제공합니다. 모든 플랫폼을 지원한다고 해도 실제 앱 품질과 기능이 플랫폼별로 다를 수 있습니다. |
| 답변 | |

#### Q6. Bending Spoons 인수 후 서비스 로드맵과 엔터프라이즈 지원 계획은?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | Vimeo는 2025년 11월 Bending Spoons에 인수되었습니다. 새 소유주 아래에서 엔터프라이즈 OTT 서비스가 어떻게 발전할지, 기존 고객에 대한 지원이 유지될지 불확실합니다. |
| 기술 배경 | 인수 후 서비스 방향이 바뀌는 것은 흔합니다. 특히 비용 절감을 위해 저수익 서비스를 종료하거나, 엔터프라이즈 지원을 축소하는 경우가 있습니다. |
| 답변 | |

#### Q12. Vimeo OTT는 플레이어 커스터마이징 외에 완전한 웹사이트 구축도 가능한가요?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | WSOPTV는 단순히 영상 플레이어만 필요한 것이 아니라, 완전한 OTT 웹사이트(홈페이지, 콘텐츠 카탈로그, 사용자 계정 관리, 검색 등)가 필요합니다. Vimeo OTT가 플레이어 임베드만 제공하는지, 아니면 전체 웹사이트까지 구축할 수 있는지 확인해야 합니다. |
| **기술 배경** | OTT 플랫폼은 크게 두 가지 방식으로 구분됩니다: (1) 플레이어 SDK만 제공 - 웹사이트는 직접 개발 필요, (2) 풀 플랫폼 제공 - 웹사이트, 앱, 백엔드까지 올인원 제공. Vimeo OTT(Streaming)는 후자를 표방하지만, 커스터마이징 범위가 WSOPTV 요구사항을 충족하는지 확인이 필요합니다. |
| **답변** | |

### 인프라/수익화 (3개)

#### Q7. 글로벌 CDN 커버리지는?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | Vimeo는 글로벌 서비스이므로 CDN 커버리지가 넓을 것으로 예상됩니다. WSOPTV의 주요 시장(북미, 유럽, 아시아)에서 지연 시간과 품질을 구체적으로 확인해야 합니다. |
| 기술 배경 | Vimeo는 자체 CDN과 함께 Fastly, Akamai 등과 파트너십을 맺고 있습니다. |
| 답변 | |

#### Q8. DRM 지원 범위는?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | WSOPTV는 유료 구독 서비스이므로 콘텐츠 보호가 필수입니다. Vimeo가 Widevine, FairPlay, PlayReady 3종 DRM을 모두 지원하는지 확인해야 합니다. |
| 기술 배경 | Vimeo는 자체 DRM 솔루션을 보유하고 있으며, Enterprise 플랜에서 "advanced DRM"을 제공합니다. |
| 답변 | |

#### Q9. 수익화 모델 유연성은? 외부 빌링 시스템(GGPoker) 연동 가능?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | Vimeo OTT는 자체 수익화 시스템(구독, 구매, 대여)을 제공하지만, GGPoker는 자체 빌링 시스템을 사용합니다. Vimeo의 내장 빌링 대신 GGPoker 빌링을 사용할 수 있는지 확인해야 합니다. |
| 기술 배경 | Vimeo OTT는 구독당 $1/월 + 거래 수수료를 부과합니다. Entitlement API로 외부 빌링 연동이 가능한지 확인이 필요합니다. |
| 답변 | |

### 연동/운영 (2개)

#### Q10. AI 기반 기능(개인화 추천, 분석)은 어느 수준?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | Vimeo는 AI 기반 개인화 추천과 고급 분석을 마케팅합니다. 이 기능이 WSOPTV의 사용자 경험 향상에 어떻게 활용될 수 있는지 확인해야 합니다. |
| 기술 배경 | AI 추천은 사용자 시청 기록을 분석하여 관심 콘텐츠를 제안합니다. |
| 답변 | |

#### Q11. 마이그레이션 지원 범위는?

| 항목 | 내용 |
|------|------|
| 왜 중요한가 | WSOPTV가 Vimeo OTT를 선택하면, 기존에 다른 플랫폼에 있던 콘텐츠와 사용자 데이터를 마이그레이션해야 할 수 있습니다. 또는 향후 Vimeo에서 다른 플랫폼으로 이전할 가능성도 고려해야 합니다. |
| 기술 배경 | 마이그레이션 지원은 콘텐츠(영상 파일, 메타데이터), 사용자 데이터(계정, 구독 정보), 분석 데이터의 이전을 포함합니다. |
| 답변 | |
