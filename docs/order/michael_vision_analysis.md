## 1. 콘텐츠 전략 해석

### 원문
> Bracelet 대회가 1년에 대략 200개 있어요.
> 50개 정도 방송을 할거 같다. (Main Event, High Roller, 등등)

### 해석

| 항목 | 내용 |
|------|------|
| 연간 대회 | 200개 Bracelet 대회 |
| 방송 대상 | 50개 대회 (25%) |
| 선정 기준 | Main Event, High Roller 등 프리미엄 대회 |

### 콘텐츠 분배

연간 약 200개 Bracelet 대회:
```
연간 약 200개 Bracelet 대회
├── WSOP VEGAS: 약 150개 (5~7월)
│   ├── ESPN: 10개 (프리미엄)
│   │   └── 독점 1년 후 WSOP TV 사용 가능
│   ├── PokerGo: 10개 (High Roller)
│   │   └── 계약 조건에 따라 이후 사용
│   └── WSOP 직접: 나머지 (동시 진행으로 일부만 방송 가능)
│       └── 즉시 WSOP TV + YouTube 동시 송출
└── 해외: 약 50개 (WSOP 직접 방송)
    ├── Super Circuit
    └── WSOP Europe

방송 제약:
- "50개 방송" = 동시 진행 제약
- 방송 가능하면 가급적 제공
- 가치 낮은 이벤트는 비효율적으로 제외
```

### OTT 솔루션 반영 사항

| 기능 | 구현 요구사항 |
|------|--------------|
| CMS 콘텐츠 분류 | `ESPN_ARCHIVE`, `POKERGO_ARCHIVE`, `WSOP_ORIGINAL` 태그 |
| 권리 관리 | 가용 시작일(Availability Date) 필드 |
| 자동 공개 | 권리 만료 시 자동 활성화 스케줄러 |

## 2. YouTube vs WSOP TV 차별화

### 원문
> YouTube: 뒤로 스크롤 안되기 + 끝나면 영상 비공개
> WSOP TV: 뒤로 가게도 할수 있게 하고 + 끝나면 영상 남아있고

### 해석

| 기능 | YouTube | WSOP TV |
|------|---------|---------|
| DVR (Timeshift) | 비활성화 | **활성화** |
| 라이브 종료 후 | 비공개 전환 | **VOD로 유지** |
| Quick VOD | 불가 | **즉시 재시청** |

### 차별화 전략

```
YouTube (무료)          WSOP TV (유료)
┌──────────────┐       ┌──────────────┐
│ 실시간만 시청  │       │ 되감기 가능   │
│ 종료 시 사라짐 │  →    │ 영상 영구 보존 │
│ 맛보기 역할   │       │ 본 서비스     │
└──────────────┘       └──────────────┘
```

### OTT 솔루션 반영 사항

| 기능 | 구현 요구사항 |
|------|--------------|
| Timeshift | 라이브 중 30분~1시간 되감기 |
| Quick VOD | 라이브 종료 즉시 VOD 자동 전환 |
| Catchup TV | 종료 후 처음부터 재시청 |

## 3. Advanced Mode 해석

### 3.1 Multi-View

#### 원문
> Multi-view 영상: 메인화면이 중앙에 있고, 각 유저들의 얼굴을 잡고 있는 화면이 옆에 또 있는 방식 (아이돌 직캠 스타일)

#### 해석

```
┌───────────────────────────────────────────┐
│                                           │
│         ┌─────────────────────┐           │
│         │                     │           │
│         │    메인 화면 (PGM)   │           │
│         │                     │           │
│         └─────────────────────┘           │
│                                           │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ │
│  │ P1  │ │ P2  │ │ P3  │ │ P4  │ │ ... │ │
│  │직캠 │ │직캠 │ │직캠 │ │직캠 │ │     │ │
│  └─────┘ └─────┘ └─────┘ └─────┘ └─────┘ │
│                                           │
└───────────────────────────────────────────┘
```

- **메인**: PD 연출 화면 (PGM)
- **서브**: 각 플레이어 개별 카메라 (최대 9명 + 보드캠)
- **인터랙션**: 클릭 시 해당 플레이어 풀스크린 전환

### 3.2 StatsView

#### 원문
> StatsView 영상: 허드같이 그 유저의 수치라든가, 플랍에서 베팅할 확률같은거, 이런게 띄어져있는 영상

#### 해석

| 구성 요소 | 설명 |
|----------|------|
| 유저 수치 | VPIP%, 3Bet%, AF 등 플레이어 통계 |
| 베팅 확률 | 플랍/턴/리버별 예상 액션 확률 |
| HUD 스타일 | 플레이어 옆에 오버레이 표시 |

### OTT 솔루션 반영 사항

| 기능 | 아키텍처 요구사항 |
|------|------------------|
| 멀티스트림 | 10개 동시 스트림 인제스트 (9 플레이어 + 1 PGM) |
| 동기화 | 모든 스트림 타임코드 동기화 |
| 데이터 API | GGPoker 플레이어 통계 API 연동 |
| 오버레이 | 클라이언트/서버 사이드 렌더링 |

## 4. 구독 모델 해석

### 원문
> $10, $50 두개 있음.
> Exclusive Content (behind-the-scenes) 는 굳이 WSOPTV 용으로 따로 제작하지 말자.
> Plus 라는건 구독이름.

### 해석

| 티어 | 가격 | 명칭 |
|------|------|------|
| Basic | $10/월 | WSOP Plus |
| Premium | $50/월 | WSOP Plus+ |

### 기능 차별화 (추정)

| 기능 | Plus ($10) | Plus+ ($50) |
|------|:----------:|:-----------:|
| 라이브 시청 | O | O |
| VOD 시청 | O | O |
| Timeshift | O | O |
| **Multi-View** | X | **O** |
| **StatsView** | X | **O** |
| 4K (향후) | X | O |

### 핵심 인사이트
- Exclusive Content 미제작 → **Advanced Mode가 유일한 차별화 포인트**
- Plus+의 가치 제안 = Multi-View + StatsView
