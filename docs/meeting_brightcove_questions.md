### 플레이어/SDK (4개)

#### Q1. Brightcove Player SDK 커스터마이징 범위는? 멀티뷰 PLP 레이아웃을 직접 구현할 수 있나요?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | WSOPTV의 핵심 차별화 기능은 멀티뷰입니다. 포커 테이블에서 메인 PGM 화면과 동시에 여러 개의 개별 플레이어 카메라를 전환하며 볼 수 있어야 합니다. |
| **기술 배경** | PLP(Picture-Layout-Picture)는 하나의 플레이어 컨테이너 안에 여러 영상을 동시에 배치하는 기술입니다. 일반적인 PIP(Picture-in-Picture)는 메인+서브 2개지만, PLP는 4개 이상의 영상을 자유롭게 배치합니다. |
| **답변** | ☐ |

#### Q2. Player v7의 플러그인 아키텍처와 제한사항은? 자체 플러그인 개발 시 빌드/배포 프로세스는?

| 항목 | 내용 |

| **기술 배경** | 플러그인 아키텍처는 기본 플레이어에 추가 기능을 붙이는 방식입니다. 잘 설계된 아키텍처는 플레이어 업데이트 시에도 플러그인이 깨지지 않고, 독립적으로 개발/배포할 수 있습니다. |
| **답변** | ☐ |

#### Q3. 자막(WebVTT) 다국어 스위칭과 실시간 자막 지원 방식은? 20개 언어 지원 시 성능 영향은?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | WSOPTV는 글로벌 서비스로 20개 언어 자막을 지원해야 합니다 (GG POKER 지원 언어 기준). 사용자가 실시간으로 언어를 전환할 때 끊김 없이 작동해야 하며, 라이브 중계 시 실시간 자막(해설)도 표시해야 합니다. |
| **기술 배경** | WebVTT는 웹 비디오 자막 표준 포맷입니다. 실시간 자막은 라이브 방송 중 해설자의 음성을 텍스트로 변환하여 표시하는 기능입니다. |
| **답변** | ☐ |

|------|------|
| **왜 중요한가** | 포커 VOD에서 사용자가 특정 핸드(예: "로열 플러시가 나온 핸드")로 바로 이동할 수 있어야 합니다. 챕터 마킹은 영상을 구간별로 나누어 이름을 붙이는 기능이고, 시크바 썸네일은 재생 바를 드래그할 때 해당 시점의 미리보기 이미지를 보여주는 기능입니다. |
| **기술 배경** | 챕터 정보는 메타데이터로 저장되며, 썸네일은 영상에서 일정 간격(예: 5초)으로 이미지를 추출하여 스프라이트 시트로 만듭니다. |

### 인프라 (3개)

#### Q9. CDN 파트너 선택 옵션과 글로벌 PoP 분포는?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | CDN(Content Delivery Network)은 전 세계 사용자에게 영상을 빠르게 전송하는 핵심 인프라입니다. CDN이 부실하면 버퍼링이 발생하고 시청 경험이 저하됩니다. |
| **기술 배경** | PoP(Point of Presence)는 CDN의 엣지 서버 위치입니다. PoP가 사용자와 가까울수록 지연 시간이 짧습니다. 주요 CDN: Akamai, CloudFront, Fastly. |
| **답변** | ☐ |

#### Q10. Multi-DRM (Widevine/FairPlay/PlayReady) 라이선스 비용 구조는?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | DRM(Digital Rights Management)은 영상 콘텐츠를 불법 복제로부터 보호하는 기술입니다. 유료 구독 서비스에서 DRM 없이는 콘텐츠가 쉽게 유출되어 수익 손실이 발생합니다. |
| **기술 배경** | 각 플랫폼별로 다른 DRM이 필요합니다. Widevine: Chrome, Android. FairPlay: Safari, iOS, Apple TV. PlayReady: Edge, Windows, Xbox. |
| **답변** | ☐ |

#### Q11. Geo-blocking 구현 방식과 VPN 우회 방지 기술은?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | 포커 콘텐츠는 지역별 라이선스 제약이 있습니다. 특정 국가에서는 도박 관련 콘텐츠 송출이 법적으로 금지되어 있어, 해당 지역 사용자의 접근을 차단해야 합니다. |
| **기술 배경** | Geo-blocking은 사용자의 IP 주소를 기반으로 위치를 파악하여 콘텐츠 접근을 제한합니다. VPN 우회 방지는 알려진 VPN IP 대역을 차단하는 기술입니다. |
| **답변** | ☐ |

#### Q13. SSO/OAuth2 연동 시 세션 관리 방식은?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | WSOPTV 사용자는 GGPass(GGPoker 통합 계정)로 로그인합니다. 별도 회원가입 없이 기존 GGPoker 계정으로 바로 시청할 수 있어야 합니다. |

#### Q14. SLA 보장 수준과 장애 발생 시 대응 프로세스는?

| 항목 | 내용 |
|------|------|
| **왜 중요한가** | WSOP Main Event 같은 대형 이벤트 중 서비스 장애가 발생하면 브랜드 이미지에 심각한 타격을 입습니다. SLA(Service Level Agreement)는 업체가 보장하는 서비스 가용성 수준입니다. |
| **기술 배경** | 가용성 99.9% = 연간 8.76시간 다운타임 허용. 99.99% = 연간 52분 다운타임 허용. MTTR(Mean Time To Recovery)은 장애 발생부터 복구까지 평균 시간입니다. |
| **답변** | ☐ |
